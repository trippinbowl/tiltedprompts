Section 01Why TiltedPrompts Is Unique

There are prompt marketplaces. There are template stores. There are MCP servers. Nobody has stitched them into a single AI-native product studio that prints its own inventory with zero marginal cost.

The Core Thesis

AI-assisted development has created a new class of builder — technical directors who orchestrate autonomous agents to ship software at unprecedented velocity. These builders need hardened infrastructure, not prototypes. TiltedPrompts exists to serve them.

Simultaneously, 63M+ Indian MSMEs are undergoing rapid digital adoption. They need automation but cannot afford custom development. TiltedPrompts serves both markets without diluting either.

5 Structural Moats

1. Zero-Cost Inventory Generation

Our OpenClaw agents generate 350 premium assets per week autonomously overnight. Every asset is validated against quality gates, HMAC-signed, and ingested directly into the production database. Competitors hire writers. We run cron jobs.

2. Dual-Market Architecture

Agent 1 speaks to Indian bakery owners in Pune. Agent 2 speaks to SaaS CTOs in San Francisco. Same infrastructure, different directives. The "Zero-Thinking Philosophy" for MSMEs and the "Framework Laws" for engineers are not just branding — they are enforced programmatically in every LLM call.

3. Production-Grade MCP (Coming March)

The MCP ecosystem is flooded with demo-grade reference implementations. TiltedMCP will ship benchmarked, monitored, SLA-backed servers. The gap between "works in a demo" and "works at scale" is the entire opportunity.

4. Indian Language Voice AI (Coming Q3)

Western voice AI platforms treat Hindi, Tamil, and Telugu as afterthoughts. TiltedVoice makes them first-class citizens with native WhatsApp Business API integration — the channel where Indian SMBs actually operate.

5. Agentic Template Design

TiltedCode templates aren't built for humans to edit. They're built for AI agents (Cursor, Claude Code, Copilot) to extend. As "vibe coding" becomes the dominant development pattern, templates designed for agentic editing beat templates designed for human editing.

Section 02Technical Architecture

Fully decoupled microservices. Every component communicates via REST APIs and can be replaced independently. Built for a founder-led team that ships with AI agents.

System Architecture

FRONTEND Next.js 16 (App Router) + React 19 + Tailwind + Framer Motion Deployed on Vercel (target: tiltedprompts.com) Dark-mode first, mobile responsive, animated DATABASE Supabase (PostgreSQL + Auth + RLS + Realtime) Tables: profiles (users), library_assets (content) Row-Level Security gates Free vs Pro content BACKEND Python Flask API (bundle_generator.py) on port 5000 LLM abstraction: Local (LM Studio/Ollama) or DeepSeek API JSON extraction from model output, retry logic AGENTS OpenClaw Agent Framework (Slack-connected) Agent 1: tilted_generator (MSME templates) Agent 2: tilted_framework_generator (Premium frameworks) Both use HMAC-SHA256 signed ingestion PAYMENTS Stripe (global) + Razorpay (India, planned) Checkout API + Webhook auto-upgrades tier FUTURE n8n (self-hosted workflow automation) TiltedMCP (Node.js/TypeScript MCP servers) TiltedVoice (WebRTC + Deepgram/ElevenLabs)

Asset Generation Pipeline

Trigger (Slack @Claw OR Cron 10PM IST) | v OpenClaw routes to correct agent (MSME or Framework) | v tools.py calls Flask API /generate | Directive injected: SIMPLICITY_DIRECTIVE (MSME) | or FRAMEWORK_DIRECTIVE (Framework) v Flask API sends to LLM (local or DeepSeek) | JSON extracted, validated v Validation Gate | MSME: No meta-prompts, [BRACKET] placeholders, min 30 chars | Framework: 3+ CAPS HEADERS, no filler, 200+ chars, [BRACKETS] v HMAC Signing (timestamp + body signed with shared secret) | v Next.js /api/ingest-bundle verifies HMAC, rate limits (20/hr) | v Supabase library_assets table (JSONB content) | v LIVE at tiltedprompts.com/members

Overnight Batch Engine (Framework Agent)

10:00 PM IST Cron triggers: python tools.py --cron Schedule: Monday = Content & Copywriting (20 sub-niches x 25 industries = 500 combos) Tuesday = Marketing & Advertising (500 combos) Wednesday = Image Gen / Midjourney (500 combos) Thursday = Sales & Cold Outreach (500 combos) Friday = Logo & Graphic Design (500 combos) Saturday = Education & Course Creation(500 combos) Sunday = Resume & Job Tools (500 combos) Per Night: 50 unique tasks sampled from 500 combos (random.sample, no repeats) Adaptive sleep: (7h * 3600 - 50 * 180s) / 49 = ~330s between generations Total: ~7 hours, 50 assets, all validated + ingested Weekly Output: 350 premium framework assets Before Repeat: 3,500 unique combos / 350 per week = 10+ weeks
